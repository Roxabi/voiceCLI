# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | ✅        |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

**Option 1 — GitHub Security Advisories (preferred):**
[Report a vulnerability](https://github.com/Roxabi/voiceCLI/security/advisories/new)

**Option 2 — Email:**
mickael@bouly.io

We aim to acknowledge within 48 hours and release a patch within 14 days for confirmed vulnerabilities.
