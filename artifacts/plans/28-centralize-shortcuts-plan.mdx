### V3 — Overlay reads labels [backend-dev + tester]

**T6** — Add `_hotkey_badge()` pure function to `overlay.py`
- **File:** `src/voicecli/overlay.py`
- **Snippet:**
  ```python
  _ABBREV = {
      "alt": "A", "shift": "Sh", "space": "Sp",
      "esc": "Esc", "tab": "Tab",
  }

  def _hotkey_badge(hotkey_str: str) -> str:
      """Convert config hotkey string to short display badge text.

      Rules:
        - 'ctrl' → 'Ctrl' when sole modifier; 'C' when combined with others
        - 'alt' → 'A', 'shift' → 'Sh', 'space' → 'Sp', 'esc' → 'Esc', 'tab' → 'Tab'
        - other parts → capitalize first letter (graceful fallback, no crash)
      """
      parts = hotkey_str.lower().split("+")
      has_other_modifier = any(p in ("alt", "shift") for p in parts)
      result = []
      for p in parts:
          if p == "ctrl":
              result.append("C" if has_other_modifier else "Ctrl")
          elif p in _ABBREV:
              result.append(_ABBREV[p])
          else:
              result.append(p.capitalize())
      return "+".join(result)
  ```
- **Verify:** `python -c "from voicecli.overlay import _hotkey_badge; print(_hotkey_badge('ctrl+space'), _hotkey_badge('alt+shift+esc'), _hotkey_badge('ctrl+shift+r'))"`
